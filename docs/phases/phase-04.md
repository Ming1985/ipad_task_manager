# Phase 04: 孩子主界面

**类型:** ui
**状态:** pending
**相关 FR:** FR-1

## 目标

创建孩子使用的主界面，展示今日任务列表，界面友好、色彩鲜艳。

## 涉及的 User Stories

- US-004: 孩子查看任务列表

- [ ] 积分余额显示区域
- [ ] 底部导航（任务、奖励、我的）

### 任务卡片设计
- [ ] 卡片组件：

## 验收标准
- [ ] 主界面显示今日待完成任务/计划
- [ ] 每个任务显示：名称、时长、积分、指定 App 图标
- [ ] 已完成任务显示勾选状态和获得积分
- [ ] 界面使用大字体、鲜艳颜色，适合儿童
- [ ] Typecheck 通过
- [ ] 使用 dev-browser skill 在模拟器中验证 UI
