## 技术决策
| 决策 | 原因 |
|-----|------|
| 使用 ScrollView + LazyVStack | 支持动态内容和流畅滚动 |
| 计划卡片展示所有任务 | 让孩子清楚看到完整任务序列 |
| 时间问候语动态计算 | 根据当前时间显示早上好/下午好等 |
| 积分显示在头部和我的页面 | 随时可见，激励完成任务 |
